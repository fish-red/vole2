Welcome to the README for the Vienna Reporter script.

What's in this directory ?
--------------------------

manifest.uuid      A checksum for the source code management tool.
manual.html        The full manual.
quick_start.html   The quickstart guide for the impatient.
readme.html        What you are reading.
vlcr.sh            The Vienna Reporter script.
vlcr.sh.sig        An OpenPGP detached signature for vlcr.sh
bugs.html          A page devoted to known bugs and issues.

Verifying the script (optional)
-------------------------------

Use your OpenPGP utility to verify the signature of vlcr.sh

I use GNU Privacy Guard like this:

gpg --verify vlcr.sh.sig

You will probably need to fetch my key from the key servers.
