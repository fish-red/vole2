Window menu

  • Activity Window

    The Activity Window displays the interaction with Cix during a blink.

  • Tasks Window

    Displays the tasks which will be performed on the next blink. It also enables you to cancel tasks that you do not want to run. This is useful if you want to cancel a messsage that you have inadvertently posted.

  • Browser

    This displays a list of Cix conferences togther with descriptions and the date a message was last posted.
