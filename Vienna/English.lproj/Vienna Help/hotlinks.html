Cix Hotlinks


cix:<number>

Selects message <number> in the current topic.

cix:<topic>:<number>

Selects message <number> in topic <topic> which is another topic of the current conference.

cix:/<topic>:<number>

Same as above

* cix: ................... Selects the first topic in the specified conference. * cix:/ .......... Parses the path to the folder and selects the last message in the folder. */
