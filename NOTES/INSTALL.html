How to install Vienna.

1.  Quit Vienna if you currently have it running.

2.  Open a Finder window for your Applications folder. Select Icon view
    for the window.

3.  Drag Vienna into your Applications folder. You may be
    asked if you wish to replace an existing copy. Click on Replace.
